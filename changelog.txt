
Version 1.3
===========
 - Removement of the old persistence layer
 - Change CreateDatebase to the CM8 tree only.
 - Redesignment of the CM8 persitence
 - Change of search perstence to EventHandler model.
 - Add of an event-handler for the LinkTableManager
 - Removement of the TypedContent construct of the datamodel classes.
 - Add the MCRMetaISO8601Date as replacement for MCRMetaDate.
 - Add the new Access Controll system as replacement for the old user privilege system.
 - Build a new small user administartion system without privileges and memberships of groups.
 - Redesign of query system.
 - Redesign of remote access. Now WebServices, OAI and Z3950 is used
 - Separate the content store from the content indexer and build it in the new search engine. 
 - Move the IBM CM8 code to a separate uninstalled module.
