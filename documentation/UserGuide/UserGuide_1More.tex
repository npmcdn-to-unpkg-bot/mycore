\subsection{Software für Unix-Systeme}
\begin{itemize}
\item {\bf Bash} (erforderlich) - Alle zusätzlichen Unix-Shell-Scripts basieren auf der Bash-Shell. Es ist also sehr sinvoll, diese auf dem System mit zu installieren, sofern diese noch nicht vorhanden ist (z. B. auf AIX).
\item {\bf CVS-Client}(erforderlich) - Dieser dient dem Check out der aktuellen MyCoRe Distribution und ist unter Linux bereits in der Distribution. Für andere Systeme muss er seperat geholt werden.
\item {\bf ANT}(erforderlich) - Unter Linux ist dies unterdessen in der Distribution enthalten, für andere Systeme muss es von Server des Apache-Projektes geholt werden.
\item {\bf xpdf} (optional) - Diese Werkzeug wird benötigt, wenn sie die 
Erzeugung von Text-Extrakten aus PDF-Dateine für die MyCoRe-Variante aus
freien Komponenten nutzen wollen. Für die Verwendung des Content Managers
ist diese Produkt nicht erforderlich, das die Funktionalität im CM enthalten 
ist.
\item {\bf ImageMagick} (optional) - Ein mächtiges Bildkonverter-Programm, welches bei der Bearbeitung großer Bildmengen sehr hilfreich ist.
\end{itemize}
Quellen sind unter anderem
\begin{itemize}
\item für einige Produkte unter AIX $\rightarrow$ \url{http://ftp.software.ibm.com/}
\item für {\bf CVS} $\rightarrow$ \url{http://cvshome.org/}
\item für {\bf ANT} $\rightarrow$ \url{http://ant.apache.org/}
\item für die freie Variante kann die entsprechende Linux-Distribution als Software-Quelle herangezogen werden
\end{itemize}
