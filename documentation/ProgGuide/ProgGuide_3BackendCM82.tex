\subsubsection{Umsetzung der Queries}
%
{\bf Hinweise!} 
\begin{itemize}
\item Strings, welche NUR Zahlen enthalten, werden als Zahl interpretiert. Läuft die Anfrage gegen ein als 'VarChar' definiertes Attribut, kommt es zum Fehler  durch den CM. Ergänzen Sie den Zahlen-String in der Suche z. B. mit einem '*' und benutzen Sie den Like Operator. Beispiel {\tt a like '0004*'}.
\end{itemize}
