\subsection{cvs-zugang}

 Der Zugang zum cvs--Server des Mycore Projekts für Entwickler erfolgt nach
 Freischaltung eines Accounts\footnote{Dieser kann über Frank Lützenkirchen
   \url{luetzenkirchen@bibl.uni-essen.de} beantragt werden} über ssh. Dem
 Freischalten sind folgende Umgebungsvariablen zu setzten:

\begin{verbatim}
CVS_RSH=ssh
CVSROOT=:ext:mcr_username@server.mycore.de:/cvs
export CVS_RSH CVSROOT
\end{verbatim}

Es empfielt sich zuerst die MyCoRe Quellen herunterzuladen. 
\begin{verbatim}
cvs -d :ext:mcr_username@server.mycore.de:/cvs checkout mycore

\end{verbatim}

Danach können sie mit 

\begin{verbatim}
cvs -d :ext:mcr_username@server.mycore.de:/cvs commit -m "Kommentar fürs
CVS" 'file' 

\end{verbatim}

die entsprechende Datei commiten.

Soll in ein bestehendes Projekt eine neue Datei integriert werden, so legt man
sie zunächst lokal im vorgesehenen (und bereits ausgecheckten!) Verzeichnis
an. Dann merkt man sie mittels \verb+cvs add <filename>+ vor. Um sie dann
global zu registrieren, erfolgt ein (verkürzt): \verb+cvs commit <filename>+
Neue Unterverzeichnisse werden auf die gleiche Weise angelegt.
